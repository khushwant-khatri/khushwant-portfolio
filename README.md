# Khushwant Khatri — Portfolio

A modern, animated React.js portfolio website.

## 🚀 Deploy to Vercel (Get a Live Link)

### Option A — Via GitHub (Recommended)

1. Create a new GitHub repo at https://github.com/new
2. Push this project:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio"
   git remote add origin https://github.com/YOUR_USERNAME/khushwant-portfolio.git
   git push -u origin main
   ```
3. Go to https://vercel.com → **New Project** → Import your repo → **Deploy**
4. ✅ Your site is live at `https://khushwant-portfolio.vercel.app`

### Option B — Vercel CLI (Fastest)

```bash
npm install
npm install -g vercel
vercel
```
Follow the prompts — live in ~60 seconds.

---

## 🛠 Run Locally

```bash
npm install
npm start
```
Open http://localhost:3000

## Build for Production

```bash
npm run build
```

## ✨ Features

- Custom animated cursor with lag effect
- Typewriter animation cycling through roles
- Floating gradient orbs in hero section
- Animated skill bars triggered on scroll
- Tabbed experience section
- Scroll reveal animations on all sections
- Responsive mobile design
- Dark theme with electric yellow accent
